1. Sororité

  • Relation : S 1 = { (Laure, Laureline ) , (Laureline, Laure ), (Agnès, Blanche ) , (Blanche, Agnès ), (Roxane, Anna ) , (Anna, Roxane ) , (Roxane, Charline ) , (Charline, Roxane ) , (Anna, Charline ) , (Charline, Anna ) }. Notons que les autres membres de la classe n’apparaissent pas dans cette relation : n’étant sœurs de personne, ils ne sont pas en relation avec qui que ce soit.

  • Champ : ℱ ( S 1 ) = { Laure , Laureline , Agnès , Blanche , Roxane , Anna , Charline } . Notons que le champ n’inclut pas les autres membres de la classe, car ℱ ( S 1 ) ne contient que les éléments en relation avec au moins un autre élément.

  • Il faut établir que ∀ a ∈ ℱ ( S 1 ) : ( a , a ) ∉ S 1 , ce qui est immédiatement visible : (Laure, Laure ) ∉ S 1 , ( Laureline , Laureline ) ∉ S 1 , ( Agnès , Agnès ) ∉ S 1 , ( Blanche , Blanche ) ∉ S 1 , ( Roxane , Roxane ) ∉ S 1 , ( Anna , Anna ) ∉ S 1 et ( Charline , Charline ) ∉ S 1 .

  • Il faut établir que ∃ a , b , c ∈ ℱ ( S 1 ) | ¬ ( a S 1 b S 1 c ⇒ a S 1 c ) , autrement dit, que ∃ a , b , c ∈ ℱ ( S 1 ) | a S 1 b S 1 c ∧ ¬ ( a S 1 c ) . Considérons par exemple a = Roxane , b = Anna , c = Roxane. Nous avons bien Roxane S 1 Anna S 1 Roxane, mais pas Roxane S 1 Roxane.

2. De mêmes parents

  • Relation : S 2 = { ( Laure , Laure ) , ( Laureline , Laureline ) , ( Laure , Laureline ) , ( Laureline , Laure ) , ( Agnès , Agnès ) , ( Blanche , Blanche ) , ( Agnès , Blanche ) , ( Blanche , Agnès ) , ( Roxane , Roxane ) , ( Anna , Anna ) , ( Charline , Charline ) , ( Roxane , Anna ) , ( Anna , Roxane ) , ( Roxane , Charline ) , ( Charline , Roxane ) , ( Anna , Charline ) , ( Charline , Anna ) , ( Abigael , Abigael ) , ( Aliosha , Aliosha ) , ( Anna-Lune , Anna-Lune ) , … }. Les points de suspension représentent toutes les paires de type ( x , x ) où x est une femme de la classe.

  • Champ : ℱ ( S 2 ) contient toutes les femmes de la classe.

  • Il faut établir que ∀ a ∈ ℱ ( S 2 ) : ( a , a ) ∈ S 2 , ce qui est immédiatement visible : Laure S 2 Laure, Laureline S 2 Laureline, etc.

  • La transitivité est satisfaite par S 2 car si a est une femme née des mêmes parents que b, et si b est une femme née des mêmes parents que c, alors a est une femme née des mêmes parents que c. (Notons que ce genre d’argumentation informelle peut se fourvoyer, comme illustre le fait qu’on aurait pu croire que la relation S 1 ci-dessus était transitive, ce qui échoue dans le cas de S 1 lorsque a = c.)

  • La relation serait non réflexive si nous avions dans la classe une famille comportant au moins un homme et une femme, mettons Olivier frère de Cécile. Dans ce cas, ( Cécile , Olivier ) ∈ S 2 , donc Olivier ∈ ℱ ( S 2 ) , mais ( Olivier , Olivier ) ∉ S 2 .

3. Gauchisme

Pour raccourcir la réponse, supposons qu’il n’y ait que six personnes et trois rangées dans notre classe : dans l’ordre de gauche à droite, rangée 1, ( Laure , Laureline , Agnès ) , rangée 2, ( Blanche , Roxane ) , et rangée 3, Charline seule.

  • Relation : G = { (Laure, Laureline ) , (Laure, Agnès ) , (Laureline, Agnès ) , (Blanche, Roxane ) }.

  • Champ : ℱ ( G ) = { Laure , Laureline , Agnès , Blanche , Roxane } . Notons que "Charline" n’apparait pas dans le champ, n’étant assise à la gauche de personne et n’ayant personne à sa gauche sur sa rangée.

  • La relation n’est pas réflexive : informellement, il existe au moins une personne qui n’est pas assise à la gauche d’elle-même ; formellement, on a bien ∃ a ∈ ℱ ( G ) | ( a , a ) ∉ G, par exemple ( Laure , Laure ) ∉ G.

  • La relation est irréflexive : informellement, personne n’est assis à la gauche de soi-même ; formellement, on a bien ∀ a ∈ ℱ ( G ) : ( a , a ) ∉ G puisque ( Laure , Laure ) ∉ G, ( Laureline , Laureline ) ∉ G, ( Agnès , Agnès ) ∉ G, ( Blanche , Blanche ) ∉ G et ( Roxane , Roxane ) ∉ G.

  • La relation est transitive : informellement, si a est assise à la gauche de b et sur la même rangée que b, et si b est assise à la gauche de c et sur la même rangée que c, alors a est effectivement assise à la gauche de c et sur la même rangée que c ; formellement, on a bien ∀ a , b , c ∈ ℱ ( G ) : ( a G b G c ⇒ a G c ) , ce qui est équivalent à ∀ a , b , c | a G b G c : a G c, puisque seul le triplet a = Laure , b = Laureline , c = Agnès satisfait a G b G c, et on a bien Laure G Agnès.

  • La relation n’est pas symétrique : formellement, on a bien ∃ a , b ∈ ℱ ( G ) | a G b ∧ ¬ ( b G a ) , par exemple Laure G Laureline mais pas Laureline G Laure.

  • La relation est asymétrique : formellement, on a bien ∀ a , b ∈ ℱ ( G ) : a G b ⇒ ¬ ( b G a ) puisque ¬ ( Laureline G Laure ) , ¬ ( Agnès G Laure ) , ¬ ( Agnès G Laureline ) et ¬ ( Roxane G Blanche ) .

  • La relation est antisymétrique : formellement, on a bien ∀ a ≠ b ∈ ℱ ( G ) : a G b ⇒ ¬ ( b G a ) , pour la même raison que pour l’asymétrie.

4. Antisymétrie : caractérisation

Prouvons qu’une relation R satisfait symétrie et antisymétrie ssi R = { ( a , a ) | a ∈ ℱ ( R ) } .

  • Considérons R = { ( a , a ) | a ∈ ℱ ( R ) } et montrons que R est symétrique. Considérons n’importe quels a , b ∈ ℱ ( R ) . Si ¬ ( a R b ) , alors l’implication a R b ⇒ b R a est satisfaite. Si a R b, alors a = b (sinon on n’aurait pas R = { ( a , a ) | a ∈ ℱ ( R ) } ) et donc b R a. Dans les deux cas, l’implication est satisfaite.

  • Considérons R = { ( a , a ) | a ∈ ℱ ( R ) } et montrons que R est antisymétrique. Considérons n’importe quels a ≠ b ∈ ℱ ( R ) . On a ¬ ( a R b ) (sinon on n’aurait pas R = { ( a , a ) | a ∈ ℱ ( R ) } ), donc l’implication a R b ⇒ ¬ ( b R a ) est satisfaite.

  • Considérons R symétrique et antisymétrique, et montrons que R = { ( a , a ) | a ∈ ℱ ( R ) } . Considérons n’importe quelle paire a R b. Par symétrie, on a b R a. Par la contraposée de l’implication d’antisymétrie a ≠ b ⇒ ¬ ( b R a ) , à savoir b R a ⇒ a = b, on obtient a = b. Ceci établit que R ⊆ { ( a , a ) | a ∈ ℱ ( R ) } . Pour établir que { ( a , a ) | a ∈ ℱ ( R ) } ⊆ R, considérons n’importe quel a ∈ ℱ ( R ) . Comme ℱ ( R ) = 𝒟 ( R ) ∪ I ( R ) , il existe b ∈ ℱ ( R ) tel que a R b ou b R a. Par symétrie, si a R b, alors b R a, et si b R a, alors a R b. Dans les deux cas, on a a R b ∧ b R a, donc par la contraposée b R a ⇒ a = b de l’implication d’antisymétrie, on obtient a = b. Donc a R a.

ⓘ
Note

La preuve ci-dessus se trouve presque mécaniquement en écrivant pour chaque partie les hypothèses, la conclusion et les définitions des termes mobilisés, et en se demandant comment obtenir la conclusion à partir des hypothèses. Si on veut aller plus loin, une preuve plus élégante peut être obtenue en évitant de décomposer l’équivalence en deux implications, ce qui rend généralement la preuve plus courte et plus agréable à lire et souvent plus informative. Une telle preuve est souvent plus difficile à trouver, c’est pourquoi elle n’est pas demandée dans le cadre de ce cours. Voici un exemple de rédaction.

Prouvons qu’une relation R satisfait symétrie et antisymétrie ssi R = { ( a , a ) | a ∈ ℱ ( R ) } . Notons d’abord que R satisfait symétrie et antisymétrie ssi ∀ a , b ∈ ℱ ( R ) : ( a R b ⇒ b R a ) ∧ ( a ≠ b ∧ a R b ⇒ ¬ ( b R a ) ) . Réécrivant l’implication d’antisymétrie en ( a R b ∧ b R a ) ⇒ a = b, on obtient que R satisfait symétrie et antisymétrie ssi ∀ a , b ∈ ℱ ( R ) : a R b ⇒ a = b. Or, ∀ a , b ∈ ℱ ( R ) : a R b ⇒ a = b ssi R = { ( a , b ) | a , b ∈ ℱ ( R ) ∧ a = b } . Notons enfin que { ( a , b ) | a , b ∈ ℱ ( R ) ∧ a = b } = { ( a , a ) | a ∈ ℱ ( R ) } .
