1. En fonction ?

Pour chacune des relations suivantes, précisez son domaine et son image et indiquez si elle est une fonction.

  1. { ( 1 , 2 ) , ( 2 , 3 ) , ( 3 , 4 ) }

  2. { ( 1 , 2 ) , ( 2 , 4 ) , ( 3 , 3 ) }

  3. { ( 1 , 2 ) , ( 1 , 3 ) , ( 2 , 3 ) , ( 3 , 3 ) , ( 4 , 3 ) , ( 5 , 3 ) , ( 6 , 3 ) , ( 7 , 3 ) , ( 8 , 3 ) , ( 9 , 3 ) , ( 10 , 3 ) }

  4. { ( x , y ) ∈ ℝ ² | y ² = x }

  5. { ( a , b ) ∈ ℕ ² | a   divise   b }

2. De gauche à droite

Donnez un exemple d’une relation qui est :

  1. droite-unique mais pas gauche-unique ;

  2. gauche-unique mais pas droite-unique ;

  3. ni l’une ni l’autre ;

  4. droite-totale sur un l’ensemble { a , b , c } .

3. Représenter une relation d’ordre

On considère la relation R sur A = { a , b , c } définie par a R b, a R c, b R c, et x R x pour tout x.

  1. Montrez que R est un ordre complet.

  2. Trouvez une fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

  3. La fonction f est-elle unique ? Expliquez.

4. Préférence et représentation

Une personne préfère x à y, noté x ≻ y, si elle choisit x lorsque les deux options sont disponibles. On observe les choix suivants : a ≻ b, a ≻ c, b ≻ c.

  1. Trouvez une fonction f : { a , b , c } → ℝ telle que x ≻ y ⇔ f ( x ) > f ( y ) .

  2. Représentez la relation ≽ associée à ces préférences, représentant x préfére à y ou x = y.

  3. Montrez qu’elle est complète et transitive.

  4. Trouvez une fonction f : { a , b , c } → ℝ telle que x ≽ y ⇔ f ( x ) ≥ f ( y ) .

5. Homomorphismes

Soit R définie par a R b, b R c, c R d, et les paires issues de la réflexivité et de la transitivité.

  1. Trouvez une fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

  2. Proposez une autre fonction g qui décrive également un homomorphisme entre R et ≥ et décrivez la transformation entre f et g.

6. Non-transitivité et impossibilité de représentation

Considérons R sur A = { a , b , c } avec a R b, b R c, mais ¬ ( a R c ) . Montrez qu’il n’existe pas de fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

  2. Soit R définie par a R b, b R c, c R d, et les paires issues de la réflexivité et de la transitivité.

  3. La relation R est-elle homomorphe à ≥ via f ? Prouvez-le.

8. Transformation admissible

  1. Donnez un exemple d’une relation R sur un ensemble A et d’une fonction f telle que R soit homomorphe à ≥ via f.

  2. Proposez une transformation φ telle que φ ∘ f décrive un autre homomorphisme à ≥.

10. Interprétation économique

Un consommateur peut détenir une télévision ou pas de télévision et un téléphone ou pas de téléphone. On représente chaque possibilité par une paire de nombres binaires, d’abord la possession de la télévision puis la possession du téléphone. La paire ( 0 , 1 ) représente pas de télévision et un téléphone, par exemple.

On considère une relation de préférence ≽ sur l’ensemble des paniers de consommation possibles A = { ( 0 , 0 ) , ( 0 , 1 ) , ( 1 , 0 ) , ( 1 , 1 ) } représentant sa préférence entre ces possibilités, sachant qu’il est indifférent entre une télévision et un téléphone.

  1. Définissez la relation ≽ en extension (c-à-d en écrivant exhaustivement ses paires de paires).

  2. Montrez que ≽ est un préordre complet.

  3. Proposez une fonction f ( x ₁ , x ₂ ) qui décrive l’homomorphisme de ≽ avec ≥.

11. Amas de biens

Un consommateur peut détenir de 0 à 10 éléphants et de 0 à 10 girafes. On représente ses détentions par une paire d’entiers, le nombre d’éléphants d’abord. Quelles que soient ses possessions, le consommateur préfère plus d’animaux, et est indifférent entre un éléphant et deux girafes, donc par exemple il est indifférent entre ( 8 , 6 ) et ( 9 , 4 ) .

  1. Définissez la relation ≽ mathématiquement, avec des conditions sur les nombres.

  2. Montrez que ≽ est un préordre complet.

12. Ordre strict

Définissons la relation P sur A = { a , b , c } par a P b, b P c et a P c.

  1. Montrez que P est transitive et irréflexive.

  2. Définissons la relation R = P ∪ { ( x , x ) | x ∈ A } . Est-ce un type de relation vue au cours ? Pourquoi (pas) ?

  3. Montrez que R est homomorphe à ≥.
