1. En fonction ?

Pour chacune des relations suivantes, indiquez si elle est une fonction. Si oui, précisez son domaine et son image.

  1. R = { 1 2 2 3 3 4 }

  2. S = { 1 2 1 3 }

  3. T = { ( x , y ) ∈ ℝ ² | y ² = x }

  4. U = { ( a , b ) | a ÷ i s e b }

2. De gauche à droite

Donnez un exemple d’une relation qui est :

  1. droite-unique mais pas gauche-unique ;

  2. gauche-unique mais pas droite-unique ;

  3. ni l’une ni l’autre ;

  4. droite-totale sur un l’ensemble { a , b , c } .

3. Représenter une relation d’ordre

On considère la relation R sur A = { a , b , c } définie par a R b, a R c, b R c, et x R x pour tout x.

  1. Montrez que R est un ordre complet.

  2. Trouvez une fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

  3. La fonction f est-elle unique ? Expliquez.

4. Préférence et représentation

Une personne préfère x à y si elle choisit x lorsque les deux options sont disponibles. On observe les choix suivants : a ≻ b, a ≻ c, b ≻ c.

  1. Représentez la relation ≽ associée à ces préférences.

  2. Montrez qu’elle est complète et transitive.

  3. Trouvez une fonction f : { a , b , c } → ℝ telle que x ≻ y ⇔ f ( x ) > f ( y ) .

5. Homomorphismes

Soit R définie par a R b, b R c, c R d, et reflexivité.

  1. Trouvez une fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

  2. Proposez une autre fonction g qui soit également homomorphe à R et décrivez la transformation entre f et g.

6. Non-transitivité et impossibilité de représentation

Considérons R sur A = { a , b , c } avec a R b, b R c, mais ¬ ( a R c ) . Montrez qu’il n’existe pas de fonction f : A → ℝ telle que a R b ⇔ f ( a ) ≥ f ( b ) .

7. Transformation admissible

  1. Donnez un exemple d’une relation R sur un ensemble A et d’une fonction f telle que R soit homomorphe à ≥ via f.

  2. Proposez une transformation φ telle que φ ∘ f définisse un autre homomorphisme à ≥.

  3. Quelle est la nature de la transformation φ (ordinale, intervalle, etc.) ?

8. Représentation et équivalence

Soit R une relation complète sur A = { x , y , z } avec x R y, y R x, x R z, z R x.

  1. Quelle est la partie symétrique de R ?

  2. Déterminez les classes d’équivalence de s y m ( R ) .

  3. Quelle est la réduction R ⋆ ?

  4. Représentez cette réduction par une fonction numérique.

9. Interprétation économique (plus difficile)

On considère une relation de préférence ≽ sur un ensemble de paniers de consommation A = { 1 0 0 1 1 1 } définie par ( x ₁ , x ₂ ) ≽ ( y ₁ , y ₂ ) ssi x ₁ + x ₂ ≥ y ₁ + y ₂.

  1. Montrez que ≽ est un préordre complet.

  2. Proposez une fonction f ( x ₁ , x ₂ ) qui la représente.

  3. Donnez un exemple de transformation admissible φ pour ce type d’échelle.

10. Difficulté supplémentaire : ordre strict

Définissons la relation P sur A = { a , b , c } par a P b, b P c, et rien d’autre.

  1. Montrez que P est transitive et irréflexive.

  2. Définissez la relation R = P ∪ { ( x , x ) | x ∈ A } . Quelle est sa nature ?

  3. Montrez que R est homomorphe à ≥.
