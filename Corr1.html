1. Animaux

1.1. Cardinal de la relation

# R 1 = 16

1.2. Champ de la relation

ℱ ( R 1 ) = { Souris , Chat , Loup , Coyote , Ours }

1.3. Cardinal du champ

# ℱ ( R 1 ) = 5

1.4. Interprétation sémantique

Toute interprétation cohérente avec la structure de la relation est acceptable, à condition qu’elle inclue bien les paires homogènes (de type ( a , a ) ). Par exemple :

  • x R 1 y ssi x n’est pas significativement plus grand que y

  • x R 1 y ssi x a approximativement la même taille ou est plus petit que y

  • x R 1 y ssi x peut être chassé par y ou x égale y

  • x R 1 y ssi x est dominé par y ou au même niveau que y dans la chaine alimentaire

1.5. Représentation graphique

Utilisons un diagramme de Hasse car R 1 est un préordre (relation réflexive et transitive).

       Souris
         |
       Chat
      /     \
     /       \
  Loup  -- Coyote
     \       /
      \     /
       Ours

Note : le diagramme de Hasse est la représentation la plus appropriée car il simplifie la visualisation : les boucles réflexives et les liens déductibles de la transitivité (Souris → Loup, Souris → Coyote, Souris → Ours et Chat → Ours) sont implicites dans le diagramme de Hasse. Le double lien entre Loup et Coyote indique qu’ils sont considérés comme équivalents par R 1 .

2. Entre deux zoos

2.1. Relation sans propriétés classiques

Par exemple, R 2 = { (Chat, Chat), (Souris, Chat) }.

  • Pas symétrique : manquent (Souris, Souris) et (Chat, Souris) pour satisfaire la symétrie

  • Pas asymétrique ni irréflexive à cause de (Chat, Chat)

  • Pas réflexive : il manque (Souris, Souris)

Note : De nombreux autres exemples sont possibles, par exemple R 2 = { (Chat, Chat), (Chat, Souris), (Souris, Chat) }.

2.2. Cardinal minimal général

Le cardinal minimal est 2 (exemple ci-dessus).

  • Avec 0 éléments : la relation ∅ est symétrique, asymétrique, réflexive et irréflexive

  • Avec 1 élément : toute relation { ( a , b ) } est réflexive si a = b et irréflexive si a ≠ b

2.3. Cardinal minimal de l’image

Le cardinal minimal de l’image est 1 (exemple ci-dessus). En effet, avec 0 éléments, donc I ( R ) = ∅, on a ℱ ( R ) = ∅, donc R = ∅, or la relation ∅ est symétrique, asymétrique, réflexive et irréflexive.

3. Les entiers

3.1. Cardinal de N

# N = 499   500. En effet, la relation N contient tous les couples ( i , j ) avec 1 ≤ i < j ≤ 1000. C’est le nombre de façons de choisir 2 éléments distincts parmi 1000 dans un ordre croissant, soit # N = C ( 1000 , 2 ) = 1000 × 999 2 = 999   000 2 .

Note : une autre réponse possible est d’observer que N = { ( 1 , 2 ) , ( 1 , 3 ) , … , ( 1 , 1000 ) } ∪ { ( 2 , 3 ) , ( 2 , 4 ) , … , ( 2 , 1000 ) } ∪ … ∪ { ( 999 , 1000 ) } et le nombre d’éléments dans chaque sous-ensemble est respectivement 999, 998, …​, 1, donc # N = 999 + 998 + … + 1 = 999 × 999 + 1 2 (intuitivement, (999 + 1)/2 est la moyenne des éléments sommés).

3.2. Antisymétrie de N

La relation N est antisymétrique : si i N j alors i < j, donc j ≮ i, donc ¬ ( j N i ) .

3.3. Transitivité de N

La relation N est transitive : si i N j et j N k, alors i < j < k, donc i < k, donc i N k.

4. Éléments imagés

4.1. Calcul de R 1 ( Souris )

R 1 ( Souris ) = { Souris , Chat , Loup , Coyote , Ours }

4.2. Calcul de E

E = { (Souris, Ours), (Ours, Souris), (Chat, Ours), (Ours, Chat), (Loup, Ours), (Ours, Loup), (Coyote, Ours), (Ours, Coyote), (Ours, Ours) }.

En effet, pour chaque animal a, R 1 ( a ) est l’ensemble des animaux au même niveau ou plus bas que a dans le diagramme de Hasse ci-dessus. Cet ensemble contient uniquement Ours si a est Ours, et sinon, donc si a est au niveau de Loup et Coyote ou plus haut (Chat ou Souris), alors l’ensemble R 1 ( a ) contient au moins Loup, Coyote et Ours.

Donc, si un des animaux d’une paire ( a , b ) est Ours, alors R 1 ( a ) ∩ R 1 ( b ) contient au plus Ours, donc son cardinal est au plus 1, donc ces paires sont dans E ; si aucun des animaux d’une paire ( a , b ) n’est Ours, alors R 1 ( a ) ∩ R 1 ( b ) contient au moins Loup, Coyote et Ours, ces paires sont donc exclues de E.

Il s’ensuit qu’une paire est dans E ssi au moins l’un des deux animaux est Ours.

Note : on peut aussi trouver la réponse par calcul exhaustif, indiqué ci-dessous. L’inconvénient de cette approche (outre son inélégance et le risque d’erreur d’inattention) est qu’elle ne met pas sur la voie pour les questions suivantes.

  • R 1 ( Souris ) = { Souris , Chat , Loup , Coyote , Ours }

  • R 1 ( Chat ) = { Chat , Loup , Coyote , Ours }

  • R 1 ( Loup ) = { Loup , Coyote , Ours }

  • R 1 ( Coyote ) = { Coyote , Loup , Ours }

  • R 1 ( Ours ) = { Ours }

  • R 1 ( Souris ) ∩ R 1 ( Souris ) = { Souris , Chat , Loup , Coyote , Ours } → cardinal 5

  • R 1 ( Souris ) ∩ R 1 ( Chat ) = { Chat , Loup , Coyote , Ours } → cardinal 4

  • R 1 ( Souris ) ∩ R 1 ( Loup ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Souris ) ∩ R 1 ( Coyote ) = { Coyote , Loup , Ours } → cardinal 3

  • R 1 ( Souris ) ∩ R 1 ( Ours ) = { Ours } → cardinal 1

  • R 1 ( Chat ) ∩ R 1 ( Chat ) = { Chat , Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Chat ) ∩ R 1 ( Loup ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Chat ) ∩ R 1 ( Coyote ) = { Coyote , Loup , Ours } → cardinal 3

  • R 1 ( Chat ) ∩ R 1 ( Ours ) = { Ours } → cardinal 1

  • R 1 ( Loup ) ∩ R 1 ( Loup ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Loup ) ∩ R 1 ( Coyote ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Loup ) ∩ R 1 ( Ours ) = { Ours } → cardinal 1

  • R 1 ( Coyote ) ∩ R 1 ( Coyote ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Coyote ) ∩ R 1 ( Loup ) = { Loup , Coyote , Ours } → cardinal 3

  • R 1 ( Coyote ) ∩ R 1 ( Ours ) = { Ours } → cardinal 1

  • R 1 ( Ours ) ∩ R 1 ( Ours ) = { Ours } → cardinal 1

  • Omettons les paires restantes, qui sont déductibles en observant que R 1 ( a ) ∩ R 1 ( b ) = R 1 ( b ) ∩ R 1 ( a ) , donc par exemple R 1 ( Loup ) ∩ R 1 ( Chat ) = R 1 ( Chat ) ∩ R 1 ( Loup ) .

Toutes les intersections impliquant deux animaux différents de Ours ont un cardinal ≥ 3.

Les seules intersections avec cardinal ≤ 1 sont celles impliquant Ours :

  • R 1 ( a ) ∩ R 1 ( Ours ) = { Ours } pour tout a ∈ ℱ ( R 1 )

  • R 1 ( Ours ) ∩ R 1 ( b ) = { Ours } pour tout b ∈ ℱ ( R 1 )

4.3. Champ de E

ℱ ( E ) = { Souris , Chat , Loup , Coyote , Ours }

Note : la simple réponse ℱ ( E ) = ℱ ( R 1 ) est également acceptable (à supposer que vous ayez répondu correctement à la Section 1.2, « Champ de la relation »).

4.4. Calcul de F

F = { ( i , j ) | i , j ∈ ⟦ 1000 ⟧ ∧ max ( i , j ) ≥ 999 }

En effet, pour tout i ∈ ⟦ 1000 ⟧, on a N ( i ) = { k | i < k ≤ 1000 } . Donc, pour deux entiers i et j, on a N ( i ) ∩ N ( j ) = { k | max ( i , j ) < k ≤ 1000 } . Cette intersection contient au plus un élément ssi max ( i , j ) ≥ 999.

Note : de multiples formulations existent, par exemple la réponse F = { ( i , j ) ∈ ⟦ 1000 ⟧ | i ≥ 999 ∨ j ≥ 999 } est également acceptable.

4.5. Calcul de ℱ ( F )

ℱ ( F ) = ⟦ 1000 ⟧.

Note : une preuve (non demandée) est que pour tout i ∈ ⟦ 1000 ⟧, on a ( i , 999 ) ∈ F.

4.6. Réflexivité de F

La relation n’est pas réflexive : par exemple, 1 ∈ ℱ ( F ) mais ( 1 , 1 ) ∉ F car max ( 1 , 1 ) = 1 < 999.

4.7. Irréflexivité de F n

F n n’est pas irréflexive, quel que soit n. En effet, ( n , n ) ∈ F n car # N ( n ) = 0 ≤ 1.

Note : cette réponse considère N définie sur ⟦ n ⟧, ce qui n’était pas explicite. Une réponse qui considère que N reste borné à 1000 quand n change est également acceptée.
