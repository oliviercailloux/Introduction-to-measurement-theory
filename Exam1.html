Consignes de réponse.

  • Répondez sur feuille blanche séparée, pas sur la feuille d’énoncé.

  • Indiquez à quelle partie d’exercice vous répondez par son numéro complet (par exemple 1.2).

Rappels et notations.

  • Le cardinal d’un ensemble A, noté # A, est le nombre d’éléments de A.

  • Étant donné n ∈ ℕ, les entiers entre 1 et n sont notés ⟦ n ⟧ = { i ∈ ℕ | 1 ≤ i ≤ n } .

  • Une relation R est asymétrique ssi ∀ a , b ∈ ℱ ( R ) : a R b ⇒ ¬ ( b R a ) .

1. Animaux (6 points)

Soit la relation R 1 = { (Souris, Chat ) , (Souris, Souris ) , (Chat, Loup ) , (Souris, Loup ), (Chat, Chat ) , (Coyote, Coyote ) , (Souris, Coyote ) , (Chat, Coyote ) , (Loup, Coyote ), (Coyote, Loup ) , (Coyote, Ours ) , (Loup, Loup ) , (Souris, Ours ) , (Loup, Ours ), (Ours, Ours ) , (Chat, Ours ) }.

  1. Donnez le cardinal de R 1 . Écrivez la réponse sous la forme # R 1 = …, il n’est pas demandé de justification. (1,5 points)

  2. Donnez le champ de R 1 . Écrivez la réponse sous la forme ℱ ( R 1 ) = …, il n’est pas demandé de justification. (1,5 points)

  3. Donnez le cardinal de ℱ ( R 1 ) . Écrivez la réponse sous la forme # ℱ ( R 1 ) = …, il n’est pas demandé de justification. (1 point)

  4. Indiquez en français ce que pourrait représenter cette relation. Écrivez la réponse sous la forme x R 1 y ssi …. (1 point)

  5. Représentez graphiquement cette relation avec la méthode la plus appropriée. Justifiez informellement et brièvement votre choix de méthode (une phrase suffit). (1 point)

2. Entre deux zoos (5 points)

  1. Donnez une relation R 2 sur un ensemble d’animaux qui n’est ni symétrique, ni asymétrique, ni réflexive, ni irréflexive. Justifiez informellement et brièvement pourquoi votre relation ne satisfait aucune de ces propriétés (une ou deux phrases en tout suffisent). (2 points)

  2. En général, quel est le cardinal minimal d’une relation qui n’est ni symétrique, ni asymétrique, ni réflexive, ni irréflexive ? Justifiez informellement et brièvement. (2 points)

  3. Quel est le cardinal minimal de l’image I ( R ) d’une relation qui n’est ni symétrique, ni asymétrique, ni réflexive, ni irréflexive ? Justifiez informellement et brièvement. (1 point)

3. Les entiers (5 points)

Considérons la relation N sur ⟦ 1000 ⟧ définie par i N j ssi i < j.

  1. Calculez # N. Écrivez la réponse sous la forme # N = …. Justifiez brièvement (informellement ou en détaillant vos calculs). (1 point)

  2. La relation N satisfait-elle l’antisymétrie ? Justifiez informellement et brièvement (une phrase suffit). (2 points)

  3. Même question pour la transitivité. (2 points)

4. Éléments imagés (4 points)

Étant donné une relation R et un élément a ∈ ℱ ( R ) , définissons l’ensemble R ( a ) = { b ∈ ℱ ( R ) | a R b } .

Les questions ci-dessous font référence aux relations R 1 et N définies dans les exercices précédents.

  1. Calculez R 1 ( S o u r i s ) . Écrivez la réponse sous la forme R 1 ( S o u r i s ) = … (en indiquant explicitement le contenu de cet ensemble), il n’est pas demandé de justification. (1 point)

  2. Calculez E = { ( a , b ) ∈ ℱ ( R 1 ) × ℱ ( R 1 ) | # ( R 1 ( a ) ∩ R 1 ( b ) ) ≤ 1 } . Écrivez la réponse sous la forme E = … (en indiquant explicitement le contenu de cet ensemble), justifiez informellement et brièvement. (0,8 point)

  3. Donnez le champ de E. Écrivez la réponse sous la forme ℱ ( E ) = …, il n’est pas demandé de justification. (0,2 point)

  4. Calculez F = { ( i , j ) ∈ ⟦ 1000 ⟧ × ⟦ 1000 ⟧ | # ( N ( i ) ∩ N ( j ) ) ≤ 1 } . Écrivez la réponse sous la forme F = { ( i , j ) | … } (en indiquant une condition simple sur i et j), justifiez informellement et brièvement. (0,8 point)

  5. Donnez le champ de F. Écrivez la réponse sous la forme ℱ ( F ) = …, il n’est pas demandé de justification. (0,2 point)

  6. La relation F est-elle réflexive ? Justifiez informellement et brièvement, puis démontrez-le formellement. (1 point)

  7. Étant donné 1 ≤ n ∈ ℕ, définissons la relation F n = { ( i , j ) ∈ ⟦ n ⟧ × ⟦ n ⟧ | # ( N ( i ) ∩ N ( j ) ) ≤ 1 } . Pour quelles valeurs de n la relation F n est-elle irréflexive ? Justifiez informellement et brièvement, puis démontrez-le formellement. (1 point bonus)
